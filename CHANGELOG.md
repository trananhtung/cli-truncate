All notable changes to this project are documented here. The format is based on

## [0.1.0] - 2026-06-21

### Added

- Initial release.
- `truncate` / `Options` — truncate to a display width with an ellipsis,
  wide-character (CJK) and ANSI-escape aware.
- `Position` (`End` / `Start` / `Middle`) and a customizable ellipsis.
- `width` — display width of a string in terminal columns (ANSI-ignored).
